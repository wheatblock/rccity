### 服务器规定
- 注册帐户须用有效邮件地址，不得转借、出售等。
- 非建筑队用户不可以在主城进行搭建。如果你想参与到建设工作中，请发主题[点这里](https://github.com/wheatblock/rccity/issues)。
- 游戏内聊天可以适当使用非规范语言乃至少量粗鄙之语，但严禁恶意攻击谩骂（尤其是问候家人者），违者可最高永久禁止游戏内聊天。
- 偶尔小熊一下可以，但是么，过分了的话全组OP乃至全服玩家将一起去熊你（手动滑稽）
